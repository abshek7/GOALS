## Deployment
If you want to access the project head over to [link](https://tinyurl.com/abhigoals) 

## Buld
You can build it through the following steps
1. Install packages
```
pip install -r requirements.txt
```
2. Run the Project
```
python app.py
```
3. `CTRL+CLICK` to access the local deployment
